PLANTILLA INVITACIÓN DIGITAL XV

Archivos principales:
- index.html: estructura y textos.
- styles.css: diseño visual.
- script.js: música, cuenta regresiva, animaciones y confeti.

Para personalizar rápido:
1. Cambia nombres, fecha, iglesia, salón y padres en index.html.
2. Cambia la fecha del evento en script.js, variable EVENT_DATE.
3. Reemplaza assets/img/hero.jpg por una foto principal.
4. Coloca tu archivo MP3 como assets/audio/musica.mp3.
5. Cambia links de Google Maps y WhatsApp en index.html.

Para publicar:
- Puedes subir toda esta carpeta a Netlify, GitHub Pages, Vercel o un hosting normal.
