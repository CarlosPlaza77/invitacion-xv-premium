PLANTILLA INVITACIÓN DIGITAL XV - V2 PREMIUM

Archivos principales:
- index.html: estructura base.
- styles.css: diseño visual responsive premium.
- script.js: música, cuenta regresiva, animaciones, galería y confeti.
- config.js: datos editables de cada evento.

Para personalizar rápido:
1. Abre config.js.
2. Cambia nombre, fecha, padres, iglesia, salón, Google Maps y WhatsApp.
3. Reemplaza assets/img/hero.jpg por la foto principal.
4. Reemplaza assets/img/foto1.jpg, foto2.jpg y foto3.jpg por fotos de galería.
5. Coloca el MP3 en assets/audio/musica.mp3.

Notas:
- Ya no necesitas editar casi nada en index.html para otro cliente.
- Para GitHub Pages, sube todos los archivos y carpetas.
- Si no hay imágenes, la plantilla muestra fondos elegantes o placeholders.
